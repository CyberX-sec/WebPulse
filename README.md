# WebPulse
WebPulse is a powerful, modular web reconnaissance and vulnerability assessment tool. It combines passive intelligence from Shodan with active scanning to uncover technologies, services, subdomains, and security issues across web infrastructure.

# Features
Dual Modes: Shodan-powered passive recon & direct active scanning
Comprehensive Scans: DNS, WHOIS, SSL certificates, cloud provider detection, subdomain enumeration, API endpoint discovery, and more
Vulnerability Detection: Checks for exposed files, misconfigurations, outdated software, and common web vulnerabilities
Structured Output: Saves results in JSON for easy analysis and reporting
Configurable: Choose scan modules and result limits to fit your needs
